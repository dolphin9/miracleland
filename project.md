# MiracleLand 工程规划

## 项目概述

MiracleLand 是奇迹师九运的官方网站，旨在介绍「奇迹之地」的背景故事、舰长 OC，以及展示粉丝二创内容。该项目定位为内容展示型网站，强调世界观呈现、角色设定与创作内容的长期扩展。

项目分为前端和后端两个部分：
- **前端**：基于 Vue.js 的响应式网站
- **后端**：WordPress Headless CMS
- **数据库**：MySQL

## 项目目标

- 提供沉浸式的世界观体验
- 展示舰长 OC 和粉丝创作
- 支持桌面端和移动端浏览
- 实现内容管理和扩展性

## 技术栈

### 前端
- Vue 3 (Composition API)
- Vite (构建工具)
- Vue Router (路由管理)
- Pinia (状态管理)
- Axios (HTTP 请求)
- HTML5 / CSS3 / JavaScript

### 后端
- WordPress (Headless CMS)
- PHP
- MySQL

### 部署环境
- Ubuntu 22.04
- 宝塔面板
- Nginx

## 功能模块

### 1. 欢迎页 (Index)
- 站点主标题与开场动画
- 主导航栏
- 视差滚动背景图
- Footer 信息

### 2. 奇迹之地介绍页
- 九运角色介绍 (图片轮播/视频 + 文字)
- 奇迹之地背景介绍 (单栏文字)

### 3. 舰长 OC 展示页
- OC 列表页 (名片展示)
- OC 详情页 (三栏布局：形象 + 故事 + 二创链接)

### 4. 二创合集页
- 分类展示 (视频、歌曲、手工、照片、绘画)
- 合集首页 (轮播图 + 公告 + 时间排序列表)
- 详情页 (媒体播放器 + 信息标签)
