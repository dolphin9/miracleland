# MiracleLand 后端文档

本文档说明 MiracleLand 项目的后端架构、技术栈和部署配置。

---

## 后端技术栈

### 核心技术
- **WordPress**（Headless CMS）
  - 版本：6.x+
  - 用作内容管理系统
- **PHP**
  - 版本：7.4+
  - 服务器端语言
- **MySQL**
  - 版本：5.7+
  - 数据库

### API 接口
- **WordPress REST API**
  - 向前端提供数据接口
  - 支持自定义端点扩展

---

## 后端功能

后端使用 WordPress 作为 Headless CMS，通过 REST API 向前端提供内容数据。

### 内容类型

- **世界观与设定**
  - 奇迹之地背景故事
  - 世界观文本内容

- **九运介绍**
  - 角色设定
  - 图片/视频资源

- **舰长 OC**
  - OC 基本信息（姓名、ID、头像）
  - OC 详细设定（故事、形象）
  - OC 相关二创关联

- **二创内容**
  - 多种类型（视频、歌曲、手工、照片、绘画）
  - 分类与标签
  - 作者、时间、简介等元数据

### 数据扩展

- 使用 PHP 进行必要的主题或接口扩展
- 数据库存储使用 MySQL
- 支持自定义字段和内容类型

---

## 部署环境

### 服务器配置

- **操作系统**：Ubuntu 22.04 LTS
- **服务器管理**：宝塔面板
- **Web 服务器**：Nginx 1.24+
- **域名与 SSL**：支持 HTTPS

### 部署架构

- **前后端分离部署**
  - 前端：静态资源托管（Netlify/Vercel）
  - 后端：WordPress 独立部署

- **API 访问**
  - 前端通过 HTTPS 访问 WordPress REST API
  - 支持跨域请求配置
