# MiracleLand 前端文档

本项目是奇迹师九运官方网站的前端部分，用于展示「奇迹之地」的背景故事、舰长 OC，以及粉丝二创内容。

项目定位为 **响应式内容展示型网站**，使用现代化的 Vue 3 技术栈，强调世界观呈现、角色设定与创作内容的沉浸式展示体验。

---

## 技术栈

- **Vue 3.5+** - 渐进式 JavaScript 框架
- **Vite 7.2+** - 下一代前端构建工具
- **Vue Router 4.6+** - 官方路由管理器
- **Pinia**（计划中）- 状态管理
- **Axios**（计划中）- HTTP 客户端
- **HTML5 / CSS3 / JavaScript (ES6+)**

## 项目概述

本网站为 **响应式内容展示型网站**，整体分为 **1 个欢迎页 + 3 个主要页面**，完美适配桌面端与移动端浏览。

---

### 页面结构总览

- 欢迎页（Index）
- 奇迹之地介绍页（九运与世界观设定）
- 舰长 OC 展示页
- 二创合集页

所有页面统一遵循响应式设计规范，根据屏幕尺寸自动调整布局结构。

---


## 一、欢迎页（Index）

欢迎页作为网站入口，承担“第一印象 + 导航 + 氛围营造”的功能，主要包含以下元素：

- **站点主标题「奇迹之地」**
  - 带有开场动画效果，用于增强仪式感与记忆点
- **主导航栏**
  - 引导用户进入三个子页面
- **视差滚动背景图**
  - 作为核心视觉，强化世界观氛围
- **Footer**
  - 放置基础信息或版权说明

### 小屏适配说明
- 页面整体由横向结构调整为纵向结构
- 标题与导航优先展示
- 背景图保持视觉完整性，动画效果适当简化

---

## 二、奇迹之地介绍页（世界观 / 九运设定）

该页面用于集中展示九运与「奇迹之地」的背景设定，整体分为两个内容区域。

### 第一部分：九运角色介绍

- **布局结构**
  - 桌面端：左右双栏布局  
    - 左侧：九运设定图片轮播 or 视频播放器
    - 右侧：文字介绍
- **小屏适配说明**
  - 布局调整为纵向排列
  - 图片轮播位于上方，文字介绍位于下方

### 第二部分：奇迹之地背景介绍

- **布局结构**
  - 单栏文字内容
- **小屏适配说明**
  - 保持单栏布局
  - 调整行距与字号以提升阅读体验

---

## 三、舰长 OC 展示页

该页面用于展示舰长 OC，并支持从列表页进入详情页。

### 1. OC 列表页

- **布局结构**
  - 舰长名片列表展示
  - 名片比例约为 **6:4**
  - 每张名片包含：
    - 圆形头像
    - 舰长 ID
- **小屏适配说明**
  - 名片按单列或双列排列
  - 自动调整名片比例与间距

### 2. OC 详情页

- **布局结构**
  - 三栏布局：
    - 左栏：舰长 OC 全身形象
    - 中栏：舰长故事与设定文字
    - 右栏：该舰长相关的二创链接集合
- **小屏适配说明**
  - 布局调整为纵向排列
  - 内容顺序为：形象图 → 故事设定 → 二创链接

---

## 四、二创合集页

该页面用于集中展示各类二创内容，并支持分类与单独详情展示。

### 1. 二创内容分类

- 视频
- 歌曲
- 手工
- 照片
- 绘画

### 2. 二创合集首页布局

#### 上半部分

- **布局结构**
  - 桌面端：两列布局  
    - 左侧：轮播图片（展示最新绘画与照片）  
    - 右侧：活动公告
- **小屏适配说明**
  - 布局调整为纵向排列
  - 轮播图与公告分行显示

#### 下半部分

- **布局结构**
  - 按时间排序的二创内容列表
- **小屏适配说明**
  - 保持单列列表结构
  - 调整间距与字体大小

### 3. 二创详情页

- **布局结构**
  - 主体内容位于页面正中  
    - 视频 / 歌曲播放器或图片
  - 下方为信息标签区
- **信息标签内容**
  - 作者
  - 分类
  - 时间
  - 简介
- **小屏适配说明**
  - 内容居中显示
  - 标签区域自动换行排列
